## AWS Cloud Project

Deployed a basic web server on AWS EC2 using Amazon Linux.

#### Steps:
- Launched EC2 instance
- Configured security group to allow HTTP traffic
- Installed Apache web server
- Hosted a simple HTML page

#### Result:

Successfully deployed a live web server accessible via public IP address.

----
## Architecture

- EC2 instance hosting web server (Apache)
- S3 bucket hosting static HTML content
- Security groups configured for HTTP access
- S3 permissions configured for public access

---
## Key Learnings

- Configured AWS EC2 instance and web server
- Integrated S3 for static content hosting
- Troubleshot and resolved S3 AccessDenied errors
- Understood differences between HTTP and HTTPS access
